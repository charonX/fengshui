# 地支

地支，又称"十二地支"，是中国传统命理学的基础概念之一。

## 十二地支

| 地支 | 五行 | 阴阳 | 生肖 | 时辰 |
|------|------|------|------|------|
| 子 | 水 | 阳 | 鼠 | 23-1 时 |
| 丑 | 土 | 阴 | 牛 | 1-3 时 |
| 寅 | 木 | 阳 | 虎 | 3-5 时 |
| 卯 | 木 | 阴 | 兔 | 5-7 时 |
| 辰 | 土 | 阳 | 龙 | 7-9 时 |
| 巳 | 火 | 阴 | 蛇 | 9-11 时 |
| 午 | 火 | 阳 | 马 | 11-13 时 |
| 未 | 土 | 阴 | 羊 | 13-15 时 |
| 申 | 金 | 阳 | 猴 | 15-17 时 |
| 酉 | 金 | 阴 | 鸡 | 17-19 时 |
| 戌 | 土 | 阳 | 狗 | 19-21 时 |
| 亥 | 水 | 阴 | 猪 | 21-23 时 |

## 地支藏干

每个地支藏有一个或多个天干：

- **子**：藏癸
- **丑**：藏己、癸、辛
- **寅**：藏甲、丙、戊
- **卯**：藏乙
- **辰**：藏戊、乙、癸
- **巳**：藏丙、戊、庚
- **午**：藏丁、己
- **未**：藏己、丁、乙
- **申**：藏庚、壬、戊
- **酉**：藏辛
- **戌**：藏戊、辛、丁
- **亥**：藏壬、甲
