# 天干

天干，又称"十天干"，是中国传统命理学的基础概念之一。

## 十天干

| 天干 | 五行 | 阴阳 | 象征 |
|------|------|------|------|
| 甲 | 木 | 阳 | 参天大树 |
| 乙 | 木 | 阴 | 花草藤蔓 |
| 丙 | 火 | 阳 | 太阳之火 |
| 丁 | 火 | 阴 | 灯烛之火 |
| 戊 | 土 | 阳 | 高山之土 |
| 己 | 土 | 阴 | 田园之土 |
| 庚 | 金 | 阳 | 刀剑之金 |
| 辛 | 金 | 阴 | 珠宝之金 |
| 壬 | 水 | 阳 | 江河之水 |
| 癸 | 水 | 阴 | 雨露之水 |

## 天干含义

**甲木**：代表生长、发展、向上，有仁德之心。

**乙木**：代表柔韧、灵活、适应性强。

**丙火**：代表热情、开朗、表现欲。

**丁火**：代表温和、细腻、思考深入。

**戊土**：代表稳重、诚信、包容。

**己土**：代表谨慎、细致、内敛。

**庚金**：代表刚毅、果断、执行力。

**辛金**：代表精致、敏感、完美主义。

**壬水**：代表智慧、流动、变化。

**癸水**：代表温柔、敏感、想象力。
