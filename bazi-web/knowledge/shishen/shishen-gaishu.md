# 十神概述

十神是八字命理中描述日主（日干）与其他干支关系的重要概念。

## 十神列表

| 十神 | 简称 | 含义 |
|------|------|------|
| 比肩 | 比 | 同性相助，兄弟姐妹 |
| 劫财 | 劫 | 异性相助，朋友同事 |
| 食神 | 食 | 我生之物，才华智慧 |
| 伤官 | 伤 | 我生之物，创意表达 |
| 偏财 | 才 | 我克之物，意外之财 |
| 正财 | 财 | 我克之物，正职收入 |
| 七杀 | 杀 | 克我之物，压力挑战 |
| 正官 | 官 | 克我之物，事业名誉 |
| 偏印 | 枭 | 生我之物，特殊才能 |
| 正印 | 印 | 生我之物，学业贵人 |

## 十神含义

**比肩劫财**：代表同辈、朋友、兄弟姐妹。比肩为同性，劫财为异性。

**食神伤官**：代表自己的才华、表达能力。食神温和，伤官外放。

**正财偏财**：代表财富、物质。正财为稳定收入，偏财为意外之财。

**正官七杀**：代表事业、名誉、压力。正官为正当职业，七杀为挑战压力。

**正印偏印**：代表学习、贵人、长辈。正印为正式学历，偏印为特殊才能。

## 十神吉凶

十神本身无绝对吉凶，需根据八字原局和日主强弱来判断：

- **身强**：喜克泄耗（财官食伤）
- **身弱**：喜生助（印星比劫）
